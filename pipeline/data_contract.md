# Data Contract (Minimal)
Check Appendix at https://agarepolayout.andyyaro.com/appendix.html

## Join key
- UEI (Unique Entity Identifier)

## Required minimum fields (example)
- UEI
- Recipient name
- Funding amount (aggregated)
- Audit signal fields used for scoring or tiering
- Risk tier (Green, Yellow, Red) or score (if applicable)

## Output naming conventions
- Use clear, stable names so dashboards and report exports do not break.
